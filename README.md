## Comment jouer ?

Dirigez le serpent rouge pour lui faire manger les proies vertes. Chaque proie consommée augmente votre score de **100 points** et allonge le serpent. La partie s'arrête si le serpent percute un mur ou s'il se mord la queue.

### Commandes au clavier :
* **`Z`** : Déplacer vers le **Haut**
* **`Q`** : Déplacer vers la **Gauche**
* **`S`** : Déplacer vers le **Bas**
* **`D`** : Déplacer vers la **Droite**
* **`B`** : Mettre le jeu en **Pause** / Reprendre
